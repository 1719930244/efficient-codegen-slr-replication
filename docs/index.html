Overview

📡
Living Resource — This companion website and the associated repository are continuously maintained. We regularly update the paper list with newly published studies on efficient LLM-based code generation. Last content update: April 2026.

Large Language Models (LLMs) have become indispensable tools for code generation, yet their computational costs—spanning data preparation, model training, inference, and evaluation—remain a critical barrier to practical adoption. This systematic review analyzes 122 primary studies to provide a comprehensive map of efficiency techniques across the full lifecycle of LLM-based code generation.

Our review spans four research questions covering data preparation (RQ1, 10 studies), model training (RQ2, 29 studies), inference optimization (RQ3, 87 studies), and evaluation practices (RQ4, 21 studies). We identify 24 fine-grained technique categories and distill 7 key findings that reveal both progress and critical gaps in the field.

Key Findings

Our systematic analysis distills 7 key findings:

Finding 1

Quality Dominates Quantity — data quality trumps volume for efficient training.

Finding 2

Minimal Parameter Updates Suffice — PEFT achieves competitive performance with a fraction of parameters.

Finding 3

What to Distill Matters More Than How — selecting the right knowledge to transfer is key.

Finding 4

Adaptive Resource Allocation Outperforms Uniform — dynamic strategies beat static resource distribution.

Finding 5

Code Structure Enables Unique Optimizations — syntactic and semantic code properties open domain-specific efficiency gains.

Finding 6

Deployment Compounds with Inference but Rarely Evaluated Together — end-to-end efficiency is under-studied.

Finding 7

Efficiency Evaluation Lags Behind Optimization — reporting standards for efficiency metrics remain inconsistent.
